## Process of The Project
- Collected and understood the Superstore sales dataset covering multiple years and business attributes.
- Cleaned and prepared the data by handling blanks, formatting dates, and creating calculated metrics.
- Built pivot tables to summarize sales, profit, quantity, and segment-wise performance.
- Designed interactive KPIs and charts with slicers for year, month, segment, and region.
- Combined all visuals into a single, user-friendly Excel dashboard for business insights.
